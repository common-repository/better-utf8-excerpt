=== better-utf8-excerpt ===
Contributors: Yang
Tags: excerpt
Stable tag: trunk

A better excerpt plugin with better support for Chinese. It also support cutomized tags.

== Description ==
该插件由 wp-utf8-excerpt 修改而来。
主要修改详见 ChangeLog.

== Installation ==
= English =
1. Unzip and upload the better-utf8-excerpt directory to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= 中文 =
1. 解压到/wp-content/plugins/目录。
2. 在插件页面激活插件。
